Proyek diStreaming
Aplikasi web streaming film sederhana. Dibuat menggunakan Laravel untuk API (Backend) dan React JS untuk antarmuka (Frontend).

Tech Stack
Frontend:

React JS

Axios (untuk koneksi API)

React Router DOM (navigasi)

Backend:

Laravel 11

Laravel Sanctum (autentikasi token)

Fitur Aplikasi
Sistem Auth: Registrasi dan Login user menggunakan token Sanctum.

Manajemen User: Tersedia fitur untuk melihat daftar semua user (List Users) dan detail profil tiap user (Single User).

Admin Panel: Halaman khusus admin untuk menambah data film baru ke database.

Keamanan: Halaman Home, Admin, dan Users diproteksi menggunakan Middleware (Protected Routes). User harus login dulu untuk bisa akses.
